2. Add `colorMode: ColorMode;` as a required field on `ITheme`, after `name` and before `groups`:
   ```typescript
   export interface ITheme {
     id: string;
     name: string;
     colorMode: ColorMode;   // 'light' for collection default; user-set for custom themes

     tokens: TokenGroup[];
     graphState?: CollectionGraphState | null;
   }
   ```
