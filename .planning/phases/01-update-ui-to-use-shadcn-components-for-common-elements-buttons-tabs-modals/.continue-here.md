<decisions_made>

- at-tabs: use slot-based `at-tab-trigger slot="tab-list"` children (not `tabs={[array]}` prop which serializes to string)
- at-tabs event: use `ref + addEventListener` not JSX `onAtuiTabChange` prop (React doesn't bind custom web component events via JSX)
- at-select: use inline `at-select-option` children (not `options={[array]}` prop)
- at-dialog open: use `data-dialog="id"` on trigger + `trigger_id="id"` on dialog (not imperative `openDialog()` ref call)
- at-dialog close (programmatic): keep `(ref.current as any)?.closeDialog?.()` — still works even though open doesn't
- Programmatic dialogs (SaveCollection, BuildTokens etc.): hidden `<button ref data-dialog>` that gets `.click()`-ed from useEffect when `isOpen` prop becomes true
- Color input: native `<input type="color">` with `style={{ width: '40px', height: '40px' }}`
- Tab content: keep React `hidden` class approach outside at-tabs (preserves form state across tab switches)

</decisions_made>
