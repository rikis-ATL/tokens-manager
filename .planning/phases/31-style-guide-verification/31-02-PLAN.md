**Checklist table with exactly 8 items:**

| # | Area | Phase 25 Decision | Steps | Expected Behavior | Result |
|---|------|-------------------|-------|-------------------|--------|

Item 1 — Color Tokens (D-07):
- Steps: "Look for color token groups in the Style Guide. Hover over individual color swatches."
- Expected: "Each color group renders as a horizontal palette row of colored swatch boxes. Hovering a swatch shows a tooltip with `token.path: resolvedHex` (e.g., `color.primary.500: #0056D2`). Swatch background matches the resolved color value."

Item 2 — Spacing Tokens (D-08):
- Steps: "Locate spacing token groups in the Style Guide."
- Expected: "Each spacing token renders as a grey bar (10px height) with width proportional to the spacing value (capped at 300px). A label next to or below each bar shows the resolved value (e.g., `16px`)."

Item 3 — Typography Tokens (D-09):
- Steps: "Locate typography token groups in the Style Guide."
- Expected: "Each typography token renders sample text with the token's font properties applied (font-family, font-size, font-weight, line-height). Token name is visible alongside the specimen."

Item 4 — Shadow Tokens (D-10):
- Steps: "Locate shadow token groups in the Style Guide."
- Expected: "Each shadow token renders as a 30x30px div with the shadow value applied as `box-shadow`. Token name is shown as a label. The shadow is visually distinguishable on the preview tile."

Item 5 — Border-Radius Tokens (D-11):
- Steps: "Locate border-radius token groups in the Style Guide."
- Expected: "Each border-radius token renders as a 30x30px div with the `border-radius` value applied. Token name is shown as a label. Different radius values produce visibly different corner rounding."

Item 6 — Other/Fallback Tokens (D-12):
- Steps: "Look for any token types that are not color, spacing, typography, shadow, or border-radius (e.g., opacity, z-index, or custom types)."
- Expected: "These tokens render as name+value text cards showing `token.path` and the resolved value. No token type is hidden — every token in the collection appears somewhere in the Style Guide."

Item 7 — Theme Switching (D-06):
- Steps: "If the collection has themes: switch between Default and a custom theme using the theme selector at the top of the page while on the Style Guide tab."
- Expected: "The Style Guide re-renders with the selected theme's token values. Color swatches change color if the theme overrides color tokens. Spacing bars resize if spacing values differ. The switch is immediate (no page reload needed)."

Item 8 — Disabled Groups (D-05):
- Steps: "If the collection has a theme with a disabled group: activate that theme and check the Style Guide."
- Expected: "Tokens from the disabled group do NOT appear in the Style Guide. Only tokens from enabled and source groups are visible. Switching back to Default shows all groups."
