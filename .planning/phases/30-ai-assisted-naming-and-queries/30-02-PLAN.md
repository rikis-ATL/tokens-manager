**A. Collection-level endpoint: `src/app/api/collections/[id]/tokens/rename-prefix/route.ts`**

Create a new file with a PATCH handler:

```typescript
// PATCH /api/collections/[id]/tokens/rename-prefix
// Body: { groupPath: string, oldPrefix: string, newPrefix: string }
```

Implementation steps:
1. `requireRole(Action.Write, params.id)` auth guard
2. Validate body: groupPath, oldPrefix, newPrefix all required strings
3. `await dbConnect()` then fetch collection with `TokenCollection.findById(params.id).lean()`
4. Extract `collection.tokens` (W3C flat object)
5. Convert to TokenGroup[] using `tokenService.processImportedTokens(rawTokens, '')`
6. Find the group matching `groupPath` — the groupPath is slash-separated (e.g. `"colors/brand"`), which matches `group.id` in the tree
7. Call `bulkReplacePrefix(groups, groupPath, oldPrefix, newPrefix)` to get updated groups
8. **Write back to DB**: Since collection.tokens is a W3C flat object, we need to convert the updated TokenGroup[] back. The simplest approach: use `$set: { tokens: updatedW3CObject }` where `updatedW3CObject` is reconstructed. However, looking at the codebase, the token service has `processImportedTokens` but no inverse. Instead, operate directly on the W3C object keys:
   - Read all keys under the group path in the W3C object
   - For each key whose leaf name starts with `oldPrefix`, rename the key by replacing `oldPrefix` with `newPrefix`
   - Use `$unset` for old keys and `$set` for new keys in a single `findByIdAndUpdate`

   **Alternative (simpler)**: Since `tokenService.processImportedTokens` gives us the tree, and themes already store `TokenGroup[]`, we could store the rebuilt tree. BUT collection.tokens is W3C format, so we must stay in that format.

   **Recommended approach**: Work directly on the W3C object keys. Given `groupPath = "colors/brand"` and `oldPrefix = "sm-"`, `newPrefix = "small-"`:
   - The W3C path is `tokens.colors.brand.sm-padding` (dot-separated)
   - Find all keys under `tokens.colors.brand` that start with `sm-`
   - Build `$unset: { "tokens.colors.brand.sm-padding": "" }` and `$set: { "tokens.colors.brand.small-padding.$value": existingValue, "tokens.colors.brand.small-padding.$type": existingType }`

   Implementation:
   ```typescript
   const tokens = (collection.tokens as Record<string, unknown>) ?? {};
   const groupDotPath = groupPath.replace(/\//g, '.');
   
   // Navigate to the group in the W3C object
   function getNestedObj(obj: Record<string, unknown>, path: string): Record<string, unknown> | null {
     const parts = path.split('.');
     let current: unknown = obj;
     for (const part of parts) {
       if (typeof current !== 'object' || current === null) return null;
       current = (current as Record<string, unknown>)[part];
     }
     return typeof current === 'object' && current !== null ? current as Record<string, unknown> : null;
   }
   
   const groupObj = getNestedObj(tokens, groupDotPath);
   if (!groupObj) return NextResponse.json({ error: 'Group not found' }, { status: 404 });
   
   const unsetFields: Record<string, string> = {};
   const setFields: Record<string, unknown> = {};
   let renamed = 0;
   
   for (const key of Object.keys(groupObj)) {
     if (key.startsWith('$')) continue; // skip $description etc.
     const val = groupObj[key] as Record<string, unknown>;
     if (typeof val !== 'object' || val === null) continue;
     const isToken = '$value' in val;
     if (isToken && key.startsWith(oldPrefix)) {
       const newKey = newPrefix + key.slice(oldPrefix.length);
       unsetFields[`tokens.${groupDotPath}.${key}`] = '';
       setFields[`tokens.${groupDotPath}.${newKey}.$value`] = val.$value;
       setFields[`tokens.${groupDotPath}.${newKey}.$type`] = val.$type ?? 'color';
       renamed++;
     }
   }
   
   if (renamed === 0) {
     return NextResponse.json({ success: true, message: 'No tokens matched the prefix', renamed: 0 });
   }
   
   await TokenCollection.findByIdAndUpdate(params.id, { $unset: unsetFields, $set: setFields });
   broadcastTokenUpdate(params.id);
   return NextResponse.json({ success: true, message: `Renamed ${renamed} token(s)`, renamed });
   ```

9. Call `broadcastTokenUpdate(params.id)` after update
10. Return `{ success: true, message: "Renamed N token(s)", renamed: N }`
