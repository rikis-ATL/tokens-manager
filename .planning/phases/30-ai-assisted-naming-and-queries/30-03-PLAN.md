**A. Create `src/app/api/collections/[id]/themes/[themeId]/tokens/single/route.ts`**

This endpoint handles granular single-token operations on theme tokens (TokenGroup[]).

**PATCH handler** — Update a single token in a theme:
```typescript
// Body: { tokenPath: string, value: string, type?: string }
// tokenPath is slash-separated full path like "colors/brand/primary"
```

Implementation:
1. `requireRole(Action.Write, params.id)` auth guard
2. Validate body: `tokenPath` required, at least one of `value` or `type` required
3. `await dbConnect()`, fetch collection, find theme by `params.themeId`
4. Parse `tokenPath` to find the group and token:
   - Split by `/` — the last segment is the token's local `path`, everything before is the group `id`
   - Example: `"colors/brand/primary"` -> groupId = `"colors/brand"`, tokenLocalPath = `"primary"`
5. Find the group in `theme.tokens` (TokenGroup[]) by `group.id === groupId` using a recursive search
6. Find the token in `group.tokens` by `token.path === tokenLocalPath`
7. If token not found, CREATE it (this supports the AI-11 flow where update_theme_token is called for tokens that exist in collection but may not have overrides in the theme yet):
   ```typescript
   if (!existingToken) {
     group.tokens.push({
       id: tokenPath,           // full path as ID
       path: tokenLocalPath,    // local name
       value: body.value ?? '',
       type: (body.type ?? 'color') as TokenType,
     });
   } else {
     if (body.value !== undefined) existingToken.value = body.value;
     if (body.type !== undefined) existingToken.type = body.type as TokenType;
   }
   ```
8. Write back using the whole-array $set pattern (same as existing PATCH /themes/[themeId]/tokens)
9. `broadcastTokenUpdate(params.id, params.themeId)`
10. Return `{ success: true, token: { path: tokenPath, value, type } }`

**DELETE handler** — Delete a single token from a theme:
```typescript
// Body: { tokenPath: string }
```

Implementation:
1. `requireRole(Action.Write, params.id)` auth guard
2. Validate body: `tokenPath` required
3. Parse `tokenPath` same as PATCH (groupId + tokenLocalPath)
4. Find group, filter out the token: `group.tokens = group.tokens.filter(t => t.path !== tokenLocalPath)`
5. Write back, broadcast, return `{ success: true }`
