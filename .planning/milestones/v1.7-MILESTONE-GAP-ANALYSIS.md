### A. Phase 29 Requirements Detail

#### AI-11: AI can create themes with suggested token values

**What it means:**  
User says "Create a dark theme for this collection" and AI:
1. Creates a new theme
2. Suggests token values based on collection defaults (e.g., inverts colors, adjusts shadows)
3. Applies suggestions to theme tokens

**Technical approach:**
- New tool: `create_theme_with_suggestions`
- Inputs: `collectionId`, `themeName`, `colorMode`, `suggestionStrategy`
- Logic:
  - Call existing `POST /api/collections/[id]/themes`
  - Generate value suggestions (color inversion, scale multiplication, etc.)
  - Apply suggestions via `PATCH /api/collections/[id]/themes/[themeId]/tokens`

**Complexity:** Medium  
Needs value suggestion heuristics (color harmony, scale adjustments).

#### AI-12: Query tokens in natural language

**What it means:**  
User says "Which tokens use #0056D2?" and AI:
1. Parses query into filter criteria
2. Searches collection tokens
3. Returns list of matching tokens with paths

**Technical approach:**
- New tool: `query_tokens`
- Inputs: `collectionId`, `query` (natural language string)
- Logic:
  - Parse query to extract filters (color value, type, group, name pattern)
  - Search `collection.tokens` with filters
  - Return matching tokens as structured list

**Complexity:** Medium  
Needs query parsing logic (regex for hex colors, type detection, name matching).

---

#### AI-13: Request natural language bulk edits

**What it means:**  
User says "Rename all spacing tokens with sm to small" and AI:
1. Parses bulk edit request
2. Finds matching tokens
3. Applies transformation
4. Confirms result

**Technical approach:**
- New tool: `bulk_edit_tokens`
- Inputs: `collectionId`, `query` (match criteria), `action` (rename/delete/change-type)
- Logic:
  - Parse query to find matching tokens (extends AI-12 query logic)
  - Apply action using existing bulk operation helpers
  - Route through existing API (similar to bulk actions in BulkActionBar)

**Complexity:** Medium  
Leverages existing bulk action code but needs natural language → match criteria translation.

---

#### AI-14: Paste values → AI-suggested names and group structure

**What it means:**  
User pastes raw values like:
```
#0056D2
#FF6600
16px
24px
```

AI responds:
```
I see 2 colors and 2 spacing values. Suggested structure:

colors/
  brand/
    primary: #0056D2
    accent: #FF6600

spacing/
  base/
    md: 16px
    lg: 24px

Shall I create these tokens?
```

**Technical approach:**
- New tool: `suggest_tokens_from_paste`
- Inputs: `collectionId`, `pastedText`
- Logic:
  - Parse pasted text (detect types: color, dimension, number, etc.)
  - Group by type
  - Generate canonical names (e.g., "primary", "accent", "md", "lg")
  - Suggest group structure
  - Return as structured proposal (not created yet)
- Follow-up: User approves → AI calls `create_token` for each

**Complexity:** Medium-High  
Needs robust parsing (handle various formats) and naming heuristics (semantic suggestions).
