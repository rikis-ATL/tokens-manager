    **Props interface:**
    ```ts
    interface GroupBreadcrumbProps {
      groups: TokenGroup[];      // The full top-level tree (masterGroups from tokens page)
      selectedGroupId: string;   // Currently selected group ID
      onSelect: (groupId: string) => void;  // Called when an ancestor segment is clicked
    }
    ```

    **Algorithm — build ancestor chain:**
    Write a helper `findAncestors(groups: TokenGroup[], targetId: string): TokenGroup[]` that:
    1. Traverses the tree recursively.
    2. Returns the full path from root to the target group (inclusive), as a `TokenGroup[]` array in order from root to target.
    3. Returns `[]` if the group is not found.

    Example: for tree `[Brands > Brand2 > Color]` and targetId = Color's id, returns `[Brands, Brand2, Color]`.

    Implementation sketch:
    ```ts
    function findAncestors(groups: TokenGroup[], targetId: string, path: TokenGroup[] = []): TokenGroup[] {
      for (const group of groups) {
        const currentPath = [...path, group];
        if (group.id === targetId) return currentPath;
        if (group.children?.length) {
          const found = findAncestors(group.children, targetId, currentPath);
          if (found.length > 0) return found;
        }
      }
      return [];
    }
    ```

    **Display label for each ancestor:**
    Use `parseGroupPath(group.name)` to get the label array, then take the LAST segment as the display label for each group — this matches the same logic TokenGroupTree uses for FlatNode.displayLabel.

    **Render:**
    - If `!selectedGroupId` or `ancestors.length === 0`: return `null` (renders nothing — empty tree fallback per user decision).
    - Render a `<nav>` with `aria-label="breadcrumb"` containing a flex row of segments.
    - Style: `text-sm py-2 px-6 border-b border-gray-100 bg-white flex items-center gap-1` on the nav.
    - Each segment: `text-gray-500 hover:text-blue-600 hover:underline cursor-pointer text-sm` for ancestors (all except the last), rendered as a `<button>` element calling `onSelect(group.id)` on click.
    - Last segment (current group): `text-gray-800 text-sm font-medium` plain `<span>`, not clickable.
    - Separator between segments: `<span className="text-gray-400 text-sm">/</span>` — a single slash between each pair.

    **Imports needed:**
    ```ts
    'use client';
    import { TokenGroup } from '@/types';
    import { parseGroupPath } from '@/utils';
    ```

    Do NOT import `findGroupById` from utils — the local `findAncestors` function is purpose-built for this. The existing `findGroupById` in token.utils.ts only returns the node itself, not its ancestors.
